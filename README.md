# 🎬 CineStream - Movie Ticket Booking App

A beautiful and modern movie ticket booking application built with **Flutter**.

## 📱 Features

- ✅ Beautiful Splash Screen with animations
- ✅ User Authentication UI (Login/Register)
- ✅ Forgot Password Flow
- ✅ Responsive Design
- ✅ Google Fonts Integration
- ✅ Custom Widgets

## 🛠️ Tech Stack

- **Framework:** Flutter
- **Language:** Dart
- **UI Design:** Custom components with Google Fonts
